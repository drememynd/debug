The PHP Debug Output library is designed for the developer who requires the ability to output debug information. This may be a preference of a developer, or in situations where a full function debugger is not available.

The standard output will look like this:

<full file path>
<Line>::<Class>::<Method>
<Label>: <Value>

Examples:

C:\xampp\htdocs\debug\examples.php
65::tester::inTester 01:09:45:673 || 00:00:00:001
$test: My Test String
